# 📊 Play Store App Review Analysis

## 📌 Project Overview

EDA project on Google Play Store data to analyze app performance.

## 🎯 Objective

Identify factors affecting ratings, installs, and reviews.

## 📂 Dataset

* Play Store Data.csv
* User Reviews.csv

## 🛠 Tools

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Insights

* Free apps dominate
* Ratings mostly 4–4.5
* Reviews increase installs

## 🚀 Conclusion

Helps developers improve app success using data insights.
